  - Декоративные элементы (звезды с pulse)
  - Success haptic feedback
  - Градиентный круг с аватаром

- **Dashboard** - Главный экран
  - Карточка профиля с данными малыша
  - Автоматический расчет возраста
  - 4 карточки статистики (сны, рост, кормление, вехи)
  - Раздел "Последняя активность"
  - Прокрутка контента

#### Анимации:
- Плавные переходы между экранами (Motion/React)
- Spring анимация (stiffness: 300, damping: 30)
- Slide эффект при навигации
- Bounce анимация для празднования
- Pulse анимация для декоративных элементов

#### Улучшения UX:
- MainButton отключается когда выбор не сделан
- Визуальная обратная связь при выборе элементов
- Консистентный дизайн на всех экранах
- Fallback кнопки для браузера

## [1.0.0] - Telegram SDK интеграция - 2026-03-04

### ✨ Добавлено

#### Telegram интеграция:
- `index.html` - добавлен Telegram Web App SDK
- `src/main.tsx` - точка входа приложения
- `src/app/hooks/useTelegram.ts` - хуки для работы с Telegram:
  - `useTelegram()` - получение данных пользователя
  - `useTelegramMainButton()` - нативная кнопка внизу
  - `useTelegramBackButton()` - кнопка назад
  - `useHapticFeedback()` - вибрация

#### Возможности:
- MainButton заменяет обычные кнопки в Telegram
- Автозаполнение имени из профиля Telegram
- Haptic feedback на всех интерактивных элементах
- Адаптация под viewport Telegram
- Fallback кнопки для тестирования в браузере

#### Стили:
- `/src/styles/index.css` - добавлены стили для Telegram:
  - `overflow: hidden` для body
  - `100dvh` для динамического viewport
  - `overscroll-behavior: none`

#### Документация:
- `README.md` - базовая документация
- `DEPLOY.md` - гайд по деплою
- `TELEGRAM_EXAMPLES.md` - примеры использования SDK

### 🔄 Изменено:
- Screen01 - добавлен MainButton и haptic feedback
- Screen03 - добавлен MainButton и haptic feedback
- App.tsx - добавлено получение данных из Telegram
