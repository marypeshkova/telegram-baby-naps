# 📱 Описание экранов

## 🎯 Поток онбординга

```
Screen01 → Screen02 → Screen03 → Screen04 → Screen05 → Dashboard
(Дата)    (Пол)      (Имя)      (Аватар)   (Финал)    (Главная)
```

## 📅 Screen01 - Выбор даты рождения

**Элементы:**
- Заголовок: "Когда родился твой малыш?"
- Подзаголовок: "Нам нужно это, чтобы считать сны 💫"
- Прогресс-бар: 15% (1/6)
- Поле с датой (кликабельное)
- Календарь с blur эффектом (открывается снизу)
- Telegram MainButton: "Далее"

**Особенности:**
- Календарь на русском языке
- Дни недели начинаются с понедельника
- Выбранная дата подсвечивается
- Blur эффект при открытии календаря
- Haptic feedback при выборе даты

**Данные:**
- `selectedDate: Date`

## 🏠 Dashboard - Главный экран

**Элементы:**

### Шапка:
- Приветствие: "Привет, {имя Telegram пользователя}! 👋"
- Заголовок: "Главная"

### Карточка профиля:
- Аватар (цвет зависит от пола)
- Имя ребенка
- Возраст (автоматический расчет)
- Дата рождения
- Градиентный фон

### Статистика (4 карточки):
1. 😴 **Сны** - "Скоро"
2. 📊 **Рост** - "Скоро"
3. 🍼 **Кормление** - "Скоро"
4. 🎯 **Вехи** - "Скоро"

### Последняя активность:
- Пустое состояние с подсказкой

**Особенности:**
- Прокрутка страницы
- Haptic feedback на карточках
- Автоматический расчет возраста

**Данные:**
- `name: string`
- `birthDate: Date`
- `gender: "boy" | "girl" | null`
- `avatar: string`

## 🎨 Дизайн система

**Цвета:**
- Фон: `#191838`
- Карточки: `#252352`
- Акцент: `#babbff`
- Текст: `#ffffff`
- Второстепенный текст: `#8483a8`

**Шрифты:**
- Заголовки: 32px, medium
- Подзаголовки: 17px, regular
- Кнопки: 17px, medium

**Эффекты:**
- Border radius: 12px-36px
- Blur: backdrop-blur
- Shadows: для выбранных элементов
- Transitions: 200-300ms

---

## 🔄 Навигация

**Переходы:**
- Анимация slide (Motion/React)
- Direction: влево при движении вперед
- Type: spring (stiffness: 300, damping: 30)

**Кнопки:**
- Telegram MainButton на всех экранах онбординга
- Fallback кнопки для браузера
- Автоматическое скрытие в Telegram
